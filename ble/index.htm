Web Bluetooth micro:bit UART

Scan Bluetooth devices (using "BBC micro:bit" as namePrefix filter). Allows to interact with the micro:bit using the Bluetooth UART service.
Example MakeCode micro:bit project here. In this example, when the micro:bit receives a string via BLE UART, it inverts it and sends it back via BLE UART.

Open console to see the messages sent from microbit.
